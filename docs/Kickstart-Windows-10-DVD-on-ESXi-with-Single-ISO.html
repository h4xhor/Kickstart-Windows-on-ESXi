Step 2 - Move ISO to Build Directory

Moves the ISO from "{ksAttuneBaseDir}" to "{ksAttuneBaseDir}/build-{targetServer.fqn}/".

Connect via ssh:

ssh {linuxattuneuser}@{attuneosbuildserver}

Execute the following script: 

                
mkdir -vp "{ksAttuneBaseDir}/build-{targetServer.fqn}"

cd "{ksAttuneBaseDir}"

mv -v "kickstart_{targetServer.fqn}.iso" "build-{targetServer.fqn}"

Step 3 - Extract Windows ISO (link)

Extracts the boot.img image file out of the WinPE ISO into the WinPE_amd64Dir directory.

Step 4 - LIN Deploy VMWare Drivers (link)

Deploys VMWare ESXi drivers to the {ksAttuneBaseDir}/build-{targetServer.fqn} folder.

Step 5 - Create Windows Boot ISO (link)
