Step 7.3 - Move ESXi Drivers to boot.wim

Execute the following script: 

                
cd "{ksAttuneBaseDir}/build-{targetServer.fqn}"

ATTUNE_DRIVERS="WinPE_BootImageDir/attune_drivers"

mkdir --parents --verbose "${ATTUNE_DRIVERS}"

DRIVERS="Drivers"
mv --verbose ${DRIVERS}/* "${ATTUNE_DRIVERS}"

rmdir --verbose "${DRIVERS}"

Step 7.4 - Move startnet.cmd to boot.wim

Execute the following script: 

                
cd {ksAttuneBaseDir}/build-{targetServer.fqn}

mv --verbose startnet.cmd \
    "WinPE_BootImageDir/Windows/System32"
